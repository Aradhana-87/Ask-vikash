🤖 Ask Vikash – College Chatbot

An interactive college chatbot website designed to provide students with quick and easy access to information like courses, fees, hostel, faculty, and college timings.

---

🚀 Features

- 📚 Course Information
- 💰 Fees Details
- 🏨 Hostel Information
- 👨‍🏫 Faculty Details
- ⏰ College Timings
- ❓ FAQ Support
- 💬 Interactive Chatbot UI

---

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: JSON-based data handling
- Tools: VS Code, Git, GitHub
