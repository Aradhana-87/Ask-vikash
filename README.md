🤖 Ask Vikash – College Chatbot

An interactive college chatbot website designed to provide students with quick and easy access to important information like courses, fees, hostel, faculty, and college timings.

---

🌟 Features

- 🎓 Course Information (BCA & BBA)
- 💰 Fees Details
- 🏠 Hostel Information
- 👩‍🏫 Faculty Details
- 🏫 College Information
- 📅 College Timings (with Sunday holiday)
- 🔍 Smart keyword-based chatbot responses
- 🎤 Voice input support (Speech Recognition)
- 📱 Responsive design (Mobile + Desktop)

---

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Data Storage: JSON files
- Browser APIs: Speech Recognition

🚀 How to Run the Project

1. Clone the repository:
   
   git clone https://github.com/Aradhana-87/Ask-vikash.git

2. Open the folder in VS Code

3. Run using Live Server
   (Right click → Open with Live Server)
